## Tools
* [Sublime Text](http://www.sublimetext.com/)
* [Atom](https://atom.io/)
* [Brackets](http://brackets.io/)

Your text editor is your tool for writing code.

* [Github Desktop](https://desktop.github.com/)
Github Desktop is how you will be uploading your code to your repository. Be sure to commit and push. *If you are comfortable with `git` using the Terminal, feel free to skip this step.* 


* [Github](https://github.com/)
Is the class website. You should have this repository synced.
Your own repository will be in a folder with the url `https://username.github.io/repositoryname`
